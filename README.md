# Sqlite_Util
Sqliteを利用する際のユーティリティツール

# 実行環境
OS：Windows11(バッチファイルのため)
SQLite：3.42.0

# 使い方
下記ダウンロードサイトからcommand-line toolsをダウンロードし、sqlite3.exeをプロジェクトルートに格納し、batchフォルダ内のbatファイルをダブルクリックする。

# sqlite3のダウンロードサイト
https://www.sqlite.org/download.html
