# Sqlite_Util
Sqliteを利用する際のユーティリティツール

# 使い方

下記ダウンロードサイトからcommand-line toolsをダウンロードし、sqlite3.exeを同じ階層に格納し、batファイルをダブルクリックする。

# sqlite3のダウンロードサイト

https://www.sqlite.org/download.html

# サポートバージョン
SQLite version 3.27.2
